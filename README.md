# Film Review System

## Overview

This project is a web-based Film Review System developed using PHP and MySQL.
Users can browse and submit reviews, while administrators can manage films, reviews, categories, and users.

## Features

* View film reviews
* Add reviews (user side)
* Contact form
* Admin dashboard with CRUD operations
* Film & category management

## Technologies Used

* PHP (PDO)
* MySQL
* HTML/CSS

## Database Setup

1. Open phpMyAdmin
2. Create a new database (e.g. film_review)
3. Import the file: /database/film_review.sql

## How to Run

1. Install XAMPP
2. Move project to:
   htdocs/film_review_system
3. Start Apache & MySQL
4. Access:
   http://localhost/film_review_system
